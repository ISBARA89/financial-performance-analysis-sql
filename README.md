# Financial & Operational Performance Analysis using SQL

## Project Overview
This project analyzes sales and cost data for a small business using SQL.

## Tools Used
- PostgreSQL
- SQL
- Git & Git Hub

## Business Questions Answered
1. Monthly revenue & profit trend
2. Most profitable product
3. Average profit per sale by product
4. Evaluation of cost drivers
5. Strategic recommendation based on data

## Key Insights
- The 5kg product generated the highest total and average profit per sale.
- Logistics costs accounted for the largest portion of expenses.
- Profitability improvement is centered around optimized logistics.

## User Acceptance Testing (UAT)
As part of the analysis, basic UAT testing was performed to ensure accuracy and alignment with business expectations.

## Activities
- Designed simple test cases to validate revenue, expense, and profit calculations.
- Verified query outputs against expected results using SQL.
- Identified and documents discrepancies for correction.

## Process Mapping

A high-level process flow was created to visualize how sales data moves from transaction to reporting, helping identify bottlenecks and improvement opportunities.
![Sales to Reporting Workflow](process-diagram/sales_to_reporting_workflow.png)

## Recommendation
Management should scale sales to 5kg and reduce logistics costs through bulk transportation and effective negotiation strategies.
